## Getting Started

Clone or download this repo first.

Install the package at the root directory using the following command:
  ```sh
  pip install -e .
  ```

You will now be able to use the package. Examples are shown in notebooks/DSP.ipynb

## Dependencies

- Numpy
- Scipy
- simpleaudio
- pretty-midi
- SoundFile
